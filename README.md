# CoralCTmini

CoralCTmini is a script-based, user-friendly, Matlab software designed to import and analyze computerized tomography (CT) scans of coral skeletal cores for density and growth variations over time. It is currently custom-built to generate a standard density curve based on the coral standards in Dr. Anne Cohen's lab, but can easily be adapted for other standards of known density.

CoralCTmini requires Matlab 2019b (or newer) and the following toolboxes:
- Image Processing Toolbox
- Signal Processing Toolbox
- Statistics and Machine Learning Toolbox
- Sensor Fusion and Tracking Toolbox
