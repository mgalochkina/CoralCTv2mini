# CoralCTv2mini

CoralCTv2mini is a script-based, user-friendly, Matlab software designed to import and analyze computerized tomography (CT) scans of coral skeletal cores for density and growth variations over time. CoralCTv2mini was written by Mariya Galochkina and Nathaniel Mollica in the Cohen Lab at Woods Hole Oceanographic Institution with funding by an NSF Graduate Research Fellowship and an MIT MathWorks Science Fellowship awarded to M. Galochkina, and is based on the work of [DeCarlo and Cohen, 2016](doi:10.5281/zenodo.57855). It is currently custom-built to generate a standard density curve based on the coral standards used in the Cohen Lab, but can easily be adapted for other standards of known density. 

CoralCTv2mini requires Matlab 2019b (or newer) and the following toolboxes:
- Image Processing Toolbox
- Signal Processing Toolbox
- Statistics and Machine Learning Toolbox
- Sensor Fusion and Tracking Toolbox
